Para cambiar imágenes:
Reemplazá los archivos dentro de assets/img manteniendo exactamente el mismo nombre. No hace falta tocar el HTML.

Promociones:
Se copió el PDF que pasaste en assets/docs/Full-sale.pdf, pero no lo mostré como promoción vigente porque indica validez 01/09/2025 al 31/10/2025. En full.html hay un bloque preparado para reemplazar ese PDF por una promo actual.

Formulario:
El formulario de contacto es visual. Para que envíe datos, conectalo con Formspree, n8n, un endpoint propio o WhatsApp.

Barra de navegación:
La barra está fija arriba, ocupa todo el ancho y el contenido arranca debajo de ella para que no atraviese textos ni fotos.

Colores:
Los colores principales están al inicio de assets/css/styles.css en :root.
